# 调用示例

```
VultrClient client = new VultrClient("YOUR API KEY");
VultrResult<Instance> instanceList = client.getInstanceList();
List<Instance> list = instanceList.getList();
for (Instance instance : list) {
    System.out.println(instance);
}
```
